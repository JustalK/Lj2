# Latsuj

## Installation

For the first installation of the project
```
$ npm i
```

## Setting

For using this project, a server mongo has to be running
```
$ mongod
```

## Build

For building the dev or build version
```
$ npm run dev
$ npm run build
```

## Server

For running the server using the last build (dev or build) :
```
$ npm run server
```
